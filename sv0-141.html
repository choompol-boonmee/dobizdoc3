90 การขอเปลี่ยนอุปกรณ์ชุดใหม่ เปลี่ยนแปลงความสามารถในการให้บริการ และเปลี่ยนแปลงตำแหน่งที่ตั้งของเครื่องอำนวยความสะดวกในการเดินอากาศ ประเภทระบบการสื่อสาร ระบบช่วยการเดินอากาศ ระบบติดตามอากาศยาน และประเภทเครื่องมืออุตุนิยมวิทยาการบิน เดือน 0 1106
91 การออกและต่ออายุใบสำคัญการแต่งตั้งผู้ปฏิบัติการทดสอบความสามารถทางภาษา เดือน 4 1091
92 การออกหนังสือรับรองใบรับรอง แบบหรือใบรับรองแบบส่วนเพิ่มเติมของ ต่างประเทศเพื่อการขอใบสำคัญสมควรเดินอากาศ เดือน 4 1057
93 การขอรับหนังสืออนุญาตก่อสร้างหรือแก้ไขเปลี่ยนแปลงอาคาร หรือสิ่งปลูกสร้างอย่างอื่นหรือปลูกต้นไม้ยืนต้นภายในเขตปลอดภัยในการเดินอากาศ บริเวณใกล้เคียงสนามบิน เดือน 15 1016
94 การเห็นชอบแผนรักษาความปลอดภัยของผู้ดำเนินการเดินอากาศ (AOSP) กรณีการออกและการต่อใบรับรองผู้ดำเนินการเดินอากาศ เดือน 0 1112
95 การรับรองแผนการฝึกอบรมพนักงานหรือนายช่างของหน่วยซ่อม (Training Programme Manual Approval) เดือน 0 1085
96 การออกใบอนุญาตประกอบกิจการบำรุงรักษาเฉพาะอากาศยานต่างประเทศในราชอาณาจักร (Foreign Aircraft Maintenance in Thailand License) เดือน 0 1079
97 การขอกำหนด Mode S Transponder ID Code (24-bit address) เดือน 0 1107
98 การขอและต่ออายุใบสำคัญสมควรเดินอากาศ เดือน 4 1063
99 การขอรับรอง การกำหนด/การเปลี่ยนแปลงแก้ไข Three-letter Designators, Telephony Designators เดือน 0 1037
100 การออกใบสำคัญสมควรเดินอากาศสำหรับส่งออกอากาศยาน (Export C of A) เดือน 4 1086
101 การเห็นชอบแผนรักษาความปลอดภัยสนามบิน (ASP) และคู่มือด้านการอำนวยความสะดวก (FAL) กรณีการขอใบรับรองผู้ได้รับใบรับรองการดำเนินงานสนามบินสาธารณะ (Aerodrome Certificate) เดือน 4 1117
102 การอนุญาตขยายระยะเวลาการบำรุงรักษาตามรายการอุปกรณ์ขั้นต่ำ (MEL Extension Approval) เดือน 0 1098
